JQMDP とは

JQMDP は jQuery Mobile Dynamic Page plugin の略である。

jQuery mobile は単独では動的なWebページを表示する機能をほとんど持たない。 そのためクライアント側で動的なページを作成しようとした場合、 多くのJavaScriptコードを記述する必要がある。

JQMDP はわずかなコードで jQuery mobile に動的なページを記述する事を目的としたプラグインである。

注：jQuery mobile は JavaScript 無しの環境に置いても動作する HTML の記述を推奨している。 JQMDP はその目的から JavaScript は必須としている。
